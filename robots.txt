# Идеи по повторным обращениям для линии 611

Авторский интерактивный сайт с предложениями по снижению повторных обращений на линии 611.

Идеи, предложения и промо-сайт разработаны Искандером.

Визуальная палитра использует цвета текущих официальных материалов компаний: Tele2 — `#121517` и `#00A6FF`, Altel — `#E72487`.

## Как открыть

Готовую версию можно открыть через `index.html`. Интернет-соединение не требуется.

Исходная интерактивная логика написана на TypeScript. Для пересборки:

```bash
npm install
npm run build
```

## Состав

- `index.html` — структура и содержание;
- `styles.css` — визуальная система и адаптивность;
- `src/main.ts` — исходный код интерактивных карточек, карты и сценариев;
- `script.js` — готовая браузерная сборка;
- `vercel.json` и `_headers` — защитные HTTP-заголовки для Vercel/Netlify;
- `SECURITY.md` — требования к безопасной публикации.

## Важно

Показатели, сроки и бюджеты являются предварительными ориентирами для обсуждения. Для инвестиционного обоснования потребуется discovery, техническая оценка, расчёт baseline и целевого эффекта на данных 611.

Публичная версия не должна содержать реальные записи звонков, ПДн, внутренние адреса систем, доступы и секреты. Если сайт предназначен для ограниченного круга, доступ необходимо закрыть через корпоративный SSO, Cloudflare Access или пароль платформы размещения.
